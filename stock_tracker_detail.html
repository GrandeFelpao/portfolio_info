Stock Tracker

Using an ESP32 I was able to get up to date stock quotes on an LCD display while using a potentiometer to scroll through different quotes.

Project Overview and Technical Details

This project provides a constantly updating "bedside clock" style display for monitoring real-time stock prices. The core challenge was implementing a reliable, low-power solution for asynchronous data retrieval and intuitive user navigation in an embedded environment.

The ESP32 was programmed to fetch data, convert the temporary JSON structure from the Google Sheets output, and render it into a visually accessible, scrollable menu on the LCD display.

Key Components & Data Flow

  • Microcontroller: ESP32 for robust Wi-Fi connectivity and device logic.
  • Data Source: A secured Google Sheet leveraging the `GoogleFinance` command to obtain up-to-date quotes.
  • Data Handling: The ESP32 fetches the sheet data (exported as JSON), parses it, and structures it for display.
  • User Interface: A compact LCD display for clear, concise quote presentation.
  • User Experience: A potentiometer enables intuitive, physical scrolling through the stock list, and a push button allows the user to force an immediate data fetch.
